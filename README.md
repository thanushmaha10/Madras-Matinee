# Madras Matinee

Madras Matinee is a full stack web application built to display and explore movies and shows in a clean and user-friendly interface.

The application provides a smooth browsing experience by fetching and displaying data from backend APIs and presenting it through a responsive frontend.

## ğŸ‘¤ User Features

Users can:
- View all available movies and shows  
- Browse **Now Showing** movies  
- View detailed movie information including genre, duration, rating, and cast  
- Add and remove movies from **Favourites**  
- Select show dates and available timings  
- Choose seats using an interactive **seat layout**  
- Book tickets (login required)  
- View booking history under **My Bookings**  
- View ticket details such as show time, seat numbers, and total price   

---

## ğŸ›  Admin Features

Admins can:
- Access a secure **Admin Dashboard**  
- View platform metrics including:
  - Total bookings  
  - Total revenue  
  - Active shows  
  - Total users  
- Add new movie shows by:
  - Selecting movies  
  - Setting show price  
  - Choosing show dates and multiple time slots  
- View all active shows with:
  - Show timings  
  - Total bookings per show  
  - Earnings per show  
- View a complete list of bookings including:
  - User name  
  - Movie name  
  - Show time  
  - Seat numbers  
  - Booking amount
